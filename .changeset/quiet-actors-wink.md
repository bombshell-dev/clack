Updates default keybindings to support Vim motion shortcuts and map the `escape` key to cancel (`ctrl+c`).

| alias 	| action 	|
|-------	|--------	|
| `k`   	| up     	|
| `l`   	| right  	|
| `j`   	| down   	|
| `h`   	| left   	|
| `esc` 	| cancel 	|
